# Add-on Plan: Public Feedback Sentiment & Topic Analysis Module
(在确保项目能够正常运行、完成代码清理整理后进行添加)

## Goal

Add a lightweight AI-assisted analysis module to the existing hospital-area traffic governance dashboard.

The module should analyze public feedback text related to hospital-area traffic issues, including complaints, survey comments, and social media-style reports.

Main capabilities:

- Sentiment classification with BERT
- Topic discovery with LDA
- Regional aggregation and dashboard visualization
- Integration with the existing Django + MySQL + ECharts system

---

## Suggested Module Name

`Public Opinion & Risk Analysis`

Chinese UI name can be:

`公众反馈与风险舆情分析`

---

## Data Scope

Use simulated or anonymized text data first.

Each feedback record should include:

- `id`
- `hospital_id`
- `district`
- `source_type`
  - complaint
  - survey
  - social_media
  - news
- `content`
- `created_at`
- `sentiment_label`
  - positive
  - neutral
  - negative
- `sentiment_score`
- `topic_id`
- `topic_keywords`

Example issues:

- parking difficulty
- road congestion
- ambulance access
- illegal parking
- pedestrian safety
- non-motor vehicle occupation
- unclear signs
- peak-hour traffic pressure

Create a new module:

```text
analytics/
  sentiment.py
  topic_modeling.py
  preprocessing.py
````

#### Sentiment Analysis

Use a Chinese BERT / RoBERTa sentiment model.

The first version can support batch inference:

```text
input: feedback text list
output: sentiment label + confidence score
```

#### Topic Modeling

Use LDA for topic extraction.

Pipeline:

```text
Chinese text cleaning
→ Jieba segmentation
→ stopword removal
→ LDA topic modeling
→ topic keyword extraction
```

Output:

```text
topic_id
topic_keywords
topic_distribution
```

## API Endpoints

Add REST-style endpoints for dashboard usage.

Suggested APIs:

```text
GET /api/opinion/summary
```

Returns:

* total feedback count
* negative feedback ratio
* top topics
* high-risk districts

```text
GET /api/opinion/by-district
```

Returns sentiment and topic statistics grouped by district.

```text
GET /api/opinion/trend
```

Returns sentiment trend over time.

```text
GET /api/opinion/topics
```

Returns topic list and top keywords.

```text
GET /api/opinion/feedback
```

Returns feedback records with filters:

* district
* hospital
* sentiment
* topic
* date range
